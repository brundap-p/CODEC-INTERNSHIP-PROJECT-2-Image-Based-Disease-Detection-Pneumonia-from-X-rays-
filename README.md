# CODEC-INTERNSHIP-PROJECT-2-Image-Based-Disease-Detection-Pneumonia-from-X-rays
PneumoXAI: Explainable Pneumonia Detection using Deep Learning and Grad-CAM

## Overview

PneumoXAI is an explainable AI system for detecting pneumonia from chest X-ray images. The project uses a deep learning model to classify X-ray images and Grad-CAM visualization to highlight the regions that influence the model's prediction.

## Features

* Pneumonia detection from chest X-ray images
* Explainable AI using Grad-CAM
* Interactive user interface
* Real-time prediction
* Visualization of affected lung regions

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Matplotlib
* Gradio
* Google Colab

## Dataset

Chest X-ray dataset containing Normal and Pneumonia images.

## Project Structure

* PXAI_DATA.ipynb - Data preprocessing and model training
* PneumoXAI_Final_Working_GradCAM.ipynb - Prediction and Grad-CAM visualization
* Model files
* Dataset files

## Results

The model predicts whether a chest X-ray indicates pneumonia and generates a Grad-CAM heatmap showing the important regions used for prediction.
